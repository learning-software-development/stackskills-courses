Text Tags

<a>

Anchor tags

Links to parts of pages or new pages

<a href="http://google.com" title="Google">Google</a>

target attribute

  • _self: same frame (default)
  • _blank: new frame
  • _parent: new parent browsing content
  • _top: load to top level context

Text Tags

<em>

Emphasis inside context

Style how you choose in CSS

I can't believe how hard that test was!

<strong>

Make something stand out

Doesn't change meaning of content (like <em>)

I use strong to show keywords that stand out.

Text Tags

<cite>

Reference title of outside work used.

books, songs, movies, speeches, etc.

not used to reference a person

I really enjoyed <cite>The Lord of the Rings</cite> by J.R.R. Tolkien.

I really enjoyed The Lord of the Rings by J.R.R. Tolkien.

<q>

Quote from another source

Can use cite attribute to reference source by URL

Quotation marks optional

<q cite="http://www.twitter.com/#!/therealnimoy">"Now that is logical. LLAP"</q> said Leonard Nimoy in the spirit of <cite>Star Trek</cite>.

"Now that is logical. LLAP"said Leonard Nimoy in the spirit of Star Trek.

Text Tags

<data>

Some content for reference with machine readable content

value attribute for machine content

<data value="11">Eleven</data>

Eleven

<time>

Specific data type for time and dates

Many rules for time formatting (see docs)

<time datetime="2011-01-12">January 12th, 2011</time>

January 12th, 2011

Text Tags

<code>

Fragment of computer code

class="language-php" to specify language

<code class="language-php">echo("Hi there!") </code>

echo("Hi there!")

<var>

Variable used in calulation or mathematical expression.

4 = <var>x</var> + 1

4 = x + 1

Text Tags

<span>

Tag a specific area of text for some purpose

Used commonly with styling

<br>

Line breaks as a part of content

Poems, music lyrics, mailing addresses

<sup><sub>

Subscript and superscript

This shows superscript

This shows subscript

Media Tags

<img>

images and the link to file

alt shows if image doesn't work

Use jpeg, gif, png, and bmp for now

<img src="images/heythere.png" alt="Giraffe!" />

Media Tags

<embed>

tags to wrap around plug-in

Can set height, width, type, and src

<iframe>

nested browsing content

One HTML page inside another

Parent/child relationship (remeber _parent?)

Media Tags

<video>

HTML5 embed for video

specify autoplay, buffer, size, controls, loop

Text inside videos only shows if video doesn't play

<audio>

HTML5 embed for audio

specify autoplay, buffer, controls, loop

Future lesson on implementing video and audio!

Tables

<thead>, <tbody>, <tfoot>

<tr>

<td>

<th>
