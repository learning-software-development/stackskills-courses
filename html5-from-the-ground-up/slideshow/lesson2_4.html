Forms Explained

Get vs. Post

            <form action="forms_submit.php" method="get">
              Form data here.
            </form>

GET

Get puts form data in the url as a query string

http://www.mysite.com/contact?name=Jon&age=30

Bookmarked, shared, remembered in browser history

Searches and queries

POST

Sends data in the body of the request

Not visible, sharable, or remembered by browser

Logins, form submissions, contact forms

Bigger chucks of info

Input types

<input type="text">

text

Single line of text info

textarea

Multiple lines of text

email

email input field; browser automatically validates

password

value is obscured with stars (does not encode the password)

Input types

checkbox

Select one or more options

radio

Select only one choice in group

Same name attribute to group radio buttons

submit
