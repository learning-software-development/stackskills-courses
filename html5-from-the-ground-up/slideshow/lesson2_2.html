Polyglot: Strive to achieve

“A document that uses polyglot markup is an HTML5 document which is at the same time an XML document and an HTML document, and which meets a well-defined set of constraints.”

Loose vs. Strict

HTML5 is very forgiving about what it will accept

  • checked="checked"
  • checked='checked'
  • checked=checked
  • checked
  • checked=""
  • checked=''

Use first as rule of thumb.

Rules of Thumb

  1. Lowercase tag and attribute names
  2. Double quoted attribute values
  3. Closing tags to match opening tags
  4. Trailing slash on self-contained elements

    5. <img src="#" />

Steps to Serialize

  1. Use correct MIME type
    • Setup on web server
    • Script with server side language
    • <?php header('Content-Type: application/xhtml+xml;charset=UTF-8'); ?>
  2. Speciify Doctype: <!DOCTYPE html>
  3. Specify XML namespace
    • <html xmlns="http://www.w3.org/1999/xhtml">
